# 🎰 BetAlgo - Provably Fair Gaming Bot

Tech-forward algorithmic gaming with verifiable fairness.

## ✨ Features

- 🎯 **Coinflip** - 50/50 chance, verifiable
- 🎲 **Dice** - Roll under/over target
- 🎡 **Roulette** - European roulette
- 💥 **Crash** - Multiplier crash game
- 🔐 **Provably Fair** - Every outcome verifiable
- 💰 **Virtual Wallet** - Track your balance
- 📊 **Statistics** - Win/loss tracking

## 🚀 Quick Start

1. Create bot with @BotFather
2. Clone repository
3. Add BOT_TOKEN to Railway
4. Deploy!

## 📋 Commands

- `/start` - Main menu
- `/balance` - Check balance
- `/stats` - View stats
- `/verify` - Verification info
- `/cancel` - Cancel game

## 🔐 Provably Fair

Every game uses SHA256 hashing:
- Server seed + Client seed + Nonce
- Verifiable after each game
- 100% transparent
